3. /stats - Estatísticas

Esta interface permite a consulta de estatísticas numéricas da base de dados

Os resultados serão exibidos no formato JSON, com os seguintes campos:

  • amount - Número que representa o total contabilizado para a consulta.

Consultas

  • /stats/totalRegistros - Total de registros de ocorrência na base de dados.

    • Ex.: http://gbif.sibbr.gov.br/api/stats/totalRegistros

  • /stats/totalRegistrosGeorreferenciados - Total de registros de ocorrência georreferenciados na base de dados.

    • Ex.: http://gbif.sibbr.gov.br/api/stats/totalRegistrosGeorreferenciados

  • /stats/totalRegistrosRepatriados - Total de registros de ocorrência repatriados na base de dados.

    • Ex.: http://gbif.sibbr.gov.br/api/stats/totalRegistros

  • /stats/totalPublicadores - Total de publicadores na base de dados.

    • Ex.: http://gbif.sibbr.gov.br/api/stats/totalPublicadores

  • /stats/totalRecursos - Total de recursos na base de dados.

    • Ex.: http://gbif.sibbr.gov.br/api/stats/totalRecursos

  • /stats/totalEspecies - Total de espécies na base de dados.

    • Ex.: http://gbif.sibbr.gov.br/api/stats/totalEspecies

  • /stats/totalFilo - Total de filos na base de dados.

    • Ex.: http://gbif.sibbr.gov.br/api/stats/totalFilo

  • /stats/totalClasse - Total de classes na base de dados.

    • Ex.: http://gbif.sibbr.gov.br/api/stats/totalClasse

  • /stats/totalOrdem - Total de ordens na base de dados.

    • Ex.: http://gbif.sibbr.gov.br/api/stats/totalOrdem

  • /stats/totalGenero - Total de gêneros na base de dados.

    • Ex.: http://gbif.sibbr.gov.br/api/stats/totalGenero

  • /stats/totalFamilia - Total de famílias na base de dados.

    • Ex.: http://gbif.sibbr.gov.br/api/stats/totalFamilia

3. /stats - Statistics

This interface allows querying of number statistics of the database

Results are presented on JSON format, with the following fields:

  • amount - Represents the total account for the query.

Queries

  • /stats/totalRegistros - Total occurrence records in the database.

    • Ex.: http://gbif.sibbr.gov.br/api/stats/totalRegistros

  • /stats/totalRegistrosGeorreferenciados - Total georeferenced occurrence records in the database.

    • Ex.: http://gbif.sibbr.gov.br/api/stats/totalRegistrosGeorreferenciados

  • /stats/totalRegistrosRepatriados - Total repatriated occurrence records in the database.

    • Ex.: http://gbif.sibbr.gov.br/api/stats/totalRegistros

  • /stats/totalPublicadores - Total amount of publishers in the database.

    • Ex.: http://gbif.sibbr.gov.br/api/stats/totalPublicadores

  • /stats/totalRecursos - Total amount of resources in the database.

    • Ex.: http://gbif.sibbr.gov.br/api/stats/totalRecursos

  • /stats/totalEspecies - Total species in the database.

    • Ex.: http://gbif.sibbr.gov.br/api/stats/totalEspecies

  • /stats/totalFilo - Total amount of phylum in the database.

    • Ex.: http://gbif.sibbr.gov.br/api/stats/totalFilo

  • /stats/totalClasse - Total amount of classes in the database.

    • Ex.: http://gbif.sibbr.gov.br/api/stats/totalClasse

  • /stats/totalOrdem - Total amount of orders in the database.

    • Ex.: http://gbif.sibbr.gov.br/api/stats/totalOrdem

  • /stats/totalGenero - Total amount of genus in the database.

    • Ex.: http://gbif.sibbr.gov.br/api/stats/totalGenero

  • /stats/totalFamilia - Total amount of families in the database.

    • Ex.: http://gbif.sibbr.gov.br/api/stats/totalFamilia
