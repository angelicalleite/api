3. /admin - Painel administrativo

Esta interface permite a realização de operações administrativas para a gestão da API.

Nota: Todas as consultas administrativas - com exceção da página raiz de documentação - devem fornecer um token de administrador válido como parâmetro da consulta

Consultas

  • /admin/ - Exibe esta página de documentação sobre as operações administrativas disponíveis.

    • Ex.: http://gbif.sibbr.gov.br/api/v1.2/admin?token=VALID_ADMIN_TOKEN

  • /admin/users - Exibe a lista de todos os usuários do sistema.

    • Ex.: http://gbif.sibbr.gov.br/api/v1.2/admin/users?token=VALID_ADMIN_TOKEN

  • /admin/users/authorized - Exibe a lista de todos os usuários autorizados do sistema.

    • Ex.: http://gbif.sibbr.gov.br/api/v1.2/admin/users/authorized?token=VALID_ADMIN_TOKEN

  • /admin/users/unauthorized - Exibe a lista de todos os usuários não autorizados do sistema.

    • Ex.: http://gbif.sibbr.gov.br/api/v1.2/admin/users/unauthorized?token=VALID_ADMIN_TOKEN

  • /admin/users/{id}/authorize - Altera o usuário para autorizado, permitindo que esta conta passe a ter permissão de acesso às operações básicas utilizando tokens de identificação.

    • Ex.: http://gbif.sibbr.gov.br/api/v1.2/admin/users/12/authorize?token=VALID_ADMIN_TOKEN

* * * English * * *

3. /admin - Administrative dashboard

This interface allows administrative operations to the API management.

Note: All administrative queries - except the admin root documentation - must provide a valid admin token as a query parameter

Queries

  • /admin/ - Displays this documentation page about available administrative operations.

    • Ex.: http://gbif.sibbr.gov.br/api/v1.2/admin?token=VALID_ADMIN_TOKEN

  • /admin/users - Displays a list of all system users.

    • Ex.: http://gbif.sibbr.gov.br/api/v1.2/admin/users?token=VALID_ADMIN_TOKEN

  • /admin/users/authorized - Displays a list of all authorized system users.

    • Ex.: http://gbif.sibbr.gov.br/api/v1.2/admin/users/authorized?token=VALID_ADMIN_TOKEN

  • /admin/users/unauthorized - Displays a list of all unauthorized system users.

    • Ex.: http://gbif.sibbr.gov.br/api/v1.2/admin/users/unauthorized?token=VALID_ADMIN_TOKEN

  • /admin/users/{id}/authorize - Changes the user to authorized, allowing this account to have access permissino to basic operations using identification tokens.

    • Ex.: http://gbif.sibbr.gov.br/api/v1.2/admin/users/12/authorize?token=VALID_ADMIN_TOKEN
